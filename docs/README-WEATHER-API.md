# Configuración del Widget de Clima

## 🌤️ API Gratuita - Open-Meteo

El widget de clima usa **Open-Meteo**, una API **completamente GRATUITA** que NO requiere registro ni API key.

### ✅ Ventajas:

- ✅ **100% Gratuito** - Sin necesidad de cuenta
- ✅ **Sin límites** - Uso ilimitado para aplicaciones personales
- ✅ **Sin API key** - Funciona inmediatamente
- ✅ **Cobertura global** - Funciona en todo el mundo (incluyendo CDMX)
- ✅ **Datos precisos** - Usa modelos meteorológicos de alta calidad
- ✅ **Sin seguimiento** - Respeta tu privacidad

## 🚀 Uso Inmediato

**No hay configuración necesaria.** Simplemente:

1. Conecta tu robot vía BLE
2. Ve a la pestaña **Acciones**
3. Presiona **🔄 Obtener Clima**
4. El navegador pedirá permiso para acceder a tu ubicación
5. ¡Listo! El clima se mostrará automáticamente

## 📍 Permisos Requeridos

El único requisito es **permitir el acceso a tu ubicación** cuando el navegador lo solicite.

### En Chrome/Edge:
- Aparecerá una notificación: "¿Permitir que este sitio acceda a tu ubicación?"
- Presiona **Permitir**

### En Firefox:
- Aparecerá una barra en la parte superior
- Presiona **Permitir**

### En Safari (iOS):
- Ve a Configuración → Safari → Ubicación
- Selecciona **Permitir**

## 🌍 Funciona en Todo el Mundo

La API detecta automáticamente tu ubicación y obtiene:
- 🌡️ Temperatura actual
- ☁️ Condiciones del clima
- 💧 Humedad relativa
- 📍 Nombre de tu ciudad

### Probado en:
- 🇲🇽 Ciudad de México, México
- 🇪🇸 Madrid, España  
- 🇺🇸 Nueva York, EE.UU.
- 🇦🇷 Buenos Aires, Argentina
- Y cualquier otra ubicación del planeta

## 🔒 Privacidad

- Tu ubicación **solo se usa** para obtener el clima
- **No se almacena** ni se envía a ningún servidor (excepto a Open-Meteo para obtener el clima)
- Open-Meteo **no rastrea** a los usuarios
- Todo el procesamiento es **local** en tu navegador
